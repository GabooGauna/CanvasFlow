# ADR-002 — React, TypeScript y Vite para la interfaz

## Estado

Aceptada.

## Contexto

CanvasFlow necesita una interfaz interactiva, modular y capaz de manejar tableros, drag and drop y lienzos visuales.

## Decisión

Utilizar React con TypeScript y Vite.

## Motivos

- React posee un ecosistema amplio;
- TypeScript mejora la seguridad y mantenibilidad;
- Vite ofrece un entorno moderno de desarrollo;
- el stack se integra con las librerías seleccionadas;
- coincide con los objetivos de aprendizaje.

## Consecuencias positivas

- componentes reutilizables;
- tipado estático;
- ecosistema maduro;
- buena experiencia de desarrollo.

## Consecuencias negativas

- React no impone una arquitectura por sí solo;
- el estado y los efectos requieren disciplina;
- TypeScript introduce una curva de aprendizaje inicial.
